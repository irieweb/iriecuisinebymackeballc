# Irie Cuisine By Mackeba LLC

🇯🇲 **Authentic Jamaican Food Truck | Bunnell, Florida**

This is the official GitHub Pages website for Irie Cuisine By Mackeba LLC.

## About

Irie Cuisine By Mackeba brings bold Jamaican flavors and island soul to Bunnell, Florida. From signature jerk chicken smoked over real charcoal to rich oxtails and curried goat — every dish is made with love and tradition.
